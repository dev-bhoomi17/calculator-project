# Calculator Project

A visually refined calculator application built using HTML, CSS, and JavaScript. The project focuses on clean design, smooth user interaction, and a modern interface.

## Features
- Animated gradient background for a dynamic visual experience  
- 3D-style button interactions for improved usability  
- Sound feedback on button clicks  
- Glassmorphism-inspired user interface  
- Responsive layout for different screen sizes
